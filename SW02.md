## 1. Bereichsoperatoren (Slicing)

- Zugriff auf Teile von Listen oder Strings mit `[start:stop]`
- `start` ist inklusiv, `stop` exklusiv
- Optional: `[start:stop:step]`

**Beispiel:**
```python
L = ['a', 'b', 'c', 'd', 'e']
L[1:4]       # ['b', 'c', 'd']
L[:3]        # ['a', 'b', 'c']
L[::2]       # ['a', 'c', 'e']
L[::-1]      # ['e', 'd', 'c', 'b', 'a']
```
